# Automatic Fire Fighting System in EV Vehicles

## Project Overview
This project is an Automatic Fire Fighting System designed for Electric Vehicles (EVs) to detect fire hazards and automatically activate a suppression mechanism.

## Features
- Temperature monitoring
- Smoke detection
- Automatic fire suppression
- Alarm system
- EV battery protection

## Components Used
- Arduino/ESP32
- Temperature Sensor
- Smoke Sensor
- Relay Module
- Buzzer
- Fire Suppression Mechanism

## Software Used
- Arduino IDE
- Embedded C/C++
